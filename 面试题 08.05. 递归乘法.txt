递归乘法。 写一个递归函数，不使用 * 运算符， 实现两个正整数的相乘。可以使用加号、减号、位移，但要吝啬一些。

示例1:

 输入：A = 1, B = 10
 输出：10

class Solution {
    public int multiply(int A, int B) {
        int a=Math.min(A,B);
        int b=Math.max(A,B);

        while(a>0){
            a--;
            return b+multiply(a,b);
        }
        return 0;
    }
}
