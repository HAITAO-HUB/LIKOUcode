字符串轮转。给定两个字符串s1和s2，请编写代码检查s2是否为s1旋转而成（比如，waterbottle是erbottlewat旋转后的字符串）。

示例1:

 输入：s1 = "waterbottle", s2 = "erbottlewat"
 输出：True

一个非常新奇的解法（当然是大佬的思路）

class Solution {
    public boolean isFlipedString(String s1, String s2) {
        if(s1.length() != s2.length())
        return false;

        return (s2+s2).indexOf(s1) != -1;
    }
}
